# 🌟 Overview

**VitaLearn Nexus** is an AI-powered school healthcare management platform designed to connect students, parents, teachers, and doctors through a centralized digital ecosystem.

The platform replaces traditional paper-based medical records with secure digital health passports, intelligent health analytics, automated reports, and seamless healthcare communication.

VitaLearn Nexus creates a **real-time, data-driven healthcare environment for educational institutions.**


---

# 🎯 Problem Statement

Millions of students still depend on:

- ❌ Paper-based medical records
- ❌ Delayed parent-doctor communication
- ❌ Poor vaccination tracking
- ❌ Lack of preventive healthcare monitoring
- ❌ No centralized student wellness analytics


Schools require a scalable solution that can:

- Store student medical information digitally
- Provide emergency health access
- Identify potential health risks early
- Improve collaboration between parents, teachers, and doctors

# ⚙️ Installation & Setup


## Clone Repository


```bash
git clone https://github.com/yourusername/VitaLearn-Nexus.git

cd VitaLearn-Nexus
```


---

## Install Dependencies


```bash
npm install

npm install --prefix frontend

npm install --prefix backend
```


---

# 🔐 Environment Variables


Create `.env` inside backend:


```env
PORT=5000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_ai_key

CLOUDINARY_URL=your_cloudinary_url

```

# 🌍 Deployment


VitaLearn Nexus supports unified deployment where the Express backend serves the compiled React frontend.


Deployment steps:

1. Connect repository to Render
2. Build command:

```
npm run build
```

3. Start command:

```
npm start
```
